 📊 Proyecto de Automatización de Ventas en Europa 📊

Este proyecto demuestra la creación, estructuración y extracción automatizada de datos comerciales utilizando **SQL** (SQLite) y **Python**.

## 🛠️ Tecnologías Utilizadas
* **Base de Datos:** SQLite / DBeaver (Modelado y carga de datos  de clientes ficticios VIP)
* **Lenguaje:** Python 3 (Extracción de métricas y lógica de negocio)

## 🚀 Funcionalidad del Script
El programa conecta de forma dinámica con la base de datos local, cruza las tablas de clientes y ventas con un `INNER JOIN`, filtra los clientes con un gasto superior a los 1000€ (`HAVING`) y exporta un reporte ejecutivo limpio directamente en la terminal.
